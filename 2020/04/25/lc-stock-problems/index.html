买卖股票

log(hans)
2020-04-25

通用解法

定义状态

dp[i][k][c] -> i：天数， k：最大交易次数， c：持有股票状态（0不持有，1持有）

  • 示例1： dp[6][2][0]： 第6天最多交易了2次，手中不持有股票（0）, 此时最大的收益
  • 示例2： dp[23][3][1]: 第23天最多交易了3次，手中持有股票（1），此时最大的收益
  • 最终状态： dp[n][1][0], 第n天最多进行了1此交易，手中不持有股票

    状态转移方程

  • 情形一（第i天不持有股票）: dp[i][k][0] = max(dp[i-1][k][0], dp[i-1][k][1] + prices[i]) 分别为（第i-1天就未持有股票， 第i天卖掉了股票）
  • 情形二（第i天持有股票）: dp[i][k][1] = max(dp[i-1][k][1], dp[i-1][k-1][0] - prices[i]) 分别为 (第i-1天就持有股票, 第i天买入股票)

    base case:

  • dp[-1][k][0] = 0 ,未开始，未持有股票，收益为0
  • dp[-1][k][1] = - float(“inf”) ,未开始，不可能持有股票，初始化收益为负无穷大
  • dp[i][0][0] = 0 ,相当于不能交易，收益为0
  • dp[i][0][1] = - float(“inf”) ,无交易，不可能有股票，初始化收益为负无穷大
